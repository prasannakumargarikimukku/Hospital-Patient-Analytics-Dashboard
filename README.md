# 🏥 Hospital Analytics Dashboard | Power BI

## 📌 Project Overview

This Power BI dashboard provides a comprehensive analysis of hospital operations by visualizing patient data, treatment trends, specialty performance, and healthcare service utilization.

The objective of this project is to transform raw healthcare data into actionable insights that support data-driven decision-making for hospital administrators and healthcare professionals.

---

## 🎯 Business Problem

Hospitals generate large volumes of patient and operational data every day. Without proper analysis, it becomes difficult to:

* Monitor patient inflow and outflow
* Identify high-demand specialties
* Analyze patient demographics
* Track inpatient and outpatient trends
* Improve resource allocation and operational efficiency

This dashboard addresses these challenges through interactive visualizations and KPI monitoring.
